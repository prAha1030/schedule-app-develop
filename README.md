### 구성 요소

- 유저 User
- 일정 Schedule
- 댓글 Comment

### 상세 구조

#### User

- 회원가입
  - 유저명, 이메일, 비밀번호를 입력하여 회원가입
    - 중복된 이메일 가입 불가
    - 비밀번호는 암호화하여 저장
- 로그인
  - 이메일, 비밀번호를 입력하여 가입 정보와 일치 시 로그인
- 유저 조회
  - 유저 전체를 조회 가능
  - 유저 id를 통해 단건의 유저 조회 가능
- 유저 수정
  - 자신의 유저명을 수정
- 유저 삭제
  - 자신의 유저 정보를 로그아웃 및 삭제

#### Schedule

- 일정 생성
  - 제목, 내용을 입력하여 일정을 생성
- 일정 조회
  - 페이지 형식의 일정을 전체 조회 (페이지 번호, 크기 설정 가능)
  - 일정 id를 통해 단건의 일정 조회 가능 (댓글 포함)
- 일정 수정
  - 일정 id를 통해 자신의 일정 제목 수정 가능
- 일정 삭제
  - 일정 id를 통해 자신의 일정 삭제 가능

#### Comment

- 댓글 생성
  - 일정 id를 통해 검색한 일정에 대한 댓글 내용 입력하여 댓글 생성

### 주의사항

#### 입력 검증 (공백 불가)

- 유저명 : 10자 이내 제한
- 이메일 : 이메일 형식
- 비밀번호 : 8~20자 사이 제한
- 일정 제목 : 20자 이내 제한
- 일정 내용 : 200자 이내 제한
- 댓글 내용 : 200자 이내 제한

#### 로그인 시 이용 가능

- 유저 수정, 삭제
- 일정 생성, 수정, 삭제
- 댓글 생성
